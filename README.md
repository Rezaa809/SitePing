## ✨ Fitur Utama

* **Real-time Monitoring**: Mengecek status website secara otomatis dalam interval singkat.
* **Latency Tracker (Ping)**: Menampilkan respon waktu (ms) untuk memantau kecepatan server.
* **Anti-Duplicate System**: Mencegah penambahan URL yang sama agar dashboard tetap rapi.
* **Modern Rect-Square Grid**: Tampilan kartu simetris yang pas untuk layar HP maupun Desktop.
* **Offline Detection**: Indikator pintar jika koneksi internet perangkat kamu terputus.
* **Local Storage**: Data website tersimpan di perangkat, tidak memerlukan database eksternal.
